# 源码解析

## 一、源码目录结构分析
整个 `Vuex` 的源码文件非常多，我们直接看最主要的文件，即 `src` 文件夹中的内容，结构示例如下：

```js
├── src
    ├── module    // 与模块相关的操作
    │   ├── module-collection.js   // 用于收集并注册根模块以及嵌套模块
    │   └── module.js   // 定义Module类，存储模块内的一些信息，例如: state……
    │
    ├── plugins   //
    │   ├── devtool.js   //
    │   └── logger.js    //
    │
    ├── helpers.js       //
    ├── index.cjs.js     //
    ├── index.js         // 入口文件
    ├── index.mjs        //
    ├── mixin.js         //
    ├── store.js         // 核心文件，定义了Store类
    └── util.js          // 提供一些工具函数，例如: deepCopy、isPromise、isObject……
```
